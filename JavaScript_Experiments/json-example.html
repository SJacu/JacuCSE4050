In this experiment, you get to play with saving data of an object in json format. This is through usage of the JSON.stringify() and JSON.parse()

When you click "save object", the data in the feilds update the profile object on the page, and the page uses JSON.stringify() to put the object in JSON format.

CLicking "Load object" will use JSON.parse() on the storage string and converts it back into a profile object type. Any valid information refills the input feilds on the page.

CLicking "Clear object" deletes the text stroage and resets the profile back to a default state
